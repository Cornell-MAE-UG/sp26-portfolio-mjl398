For this project I was first tasked with creating a nutcracker that had the ability to crack a macadamia nut by hand. Then after designing this I was then tasked with changing my design to now be operated with a linear actuator instead of by hand. For this project I assumed the macadamia nut would crack under a load of approximately 2177 newtons and that the nut had a average radius of 3 centimeters. I also assumed the average grip strength of a human to be around 500 newtowns. 

#### Task One
For the first task I planned to use a large lever in order to amplify the input force provided by the hand to make the nut crack despite the person providing considerably less force than is needed to crack the nut. A rough schematic of this initial design can be seen above. So I solved for the required length of the lever needed to crack the nut given the input load which I found to be 17.4 centimeters. 

![Task one rough sketch]({{ "/assets/images/nutcracker/task-two.png" | relative_url }}){: .inline-image-r style="width: 250px"}

#### Task Two

For the second task where I was to add a linear actuator to provide the force I decided to minorly change my design. For my new design I decided to remove the bottom handle entirely and replace it a surface in which the top handle would be mounted to via a pivot. I then planned to add the actuator above the top handle and shown here on the right in order to provide the input force. The actuator which I selected for this project provided a maximum of 752 newtons and was the IP65 mini linerar actuator, which can be found at the link below. I was also able to shrink the size of this handle as the actuator provides more force than a person could. Once I modified my design and did my calculations again I found that the lever could be 11.6 centimeters, which meant that adding this actuator allowed me to shrink the handle by about one third of its original length. After completing this project I think that the second iteration could be made more compact by moving the actuator under or next to the lever in some way. This change would also allow this design to be more usable as in this case it being more compact would make it significantly more useful.
